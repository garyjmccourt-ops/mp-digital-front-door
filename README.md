# Mermaid Pools Digital Front Door

Historical reference only
Not an active client project
Reusable pattern extracted into thebrightergroup/brightsites-trade-template

## Project summary

Mermaid Pools Digital Front Door is the first-stage digital foundation for Mermaid Pools.

It is not intended to be only a brochure website refresh. The project should create a modern, premium website and structured customer enquiry pathway that helps Mermaid Pools increase qualified enquiries, showcase completed projects, and improve visibility of enquiry quality, source, status and conversion.

The build should avoid becoming a locked brochure website. It should be structured so later digital tools can be added without rebuilding from scratch.

## Core objective

Build Mermaid Pools’ Digital Front Door: a premium website, structured enquiry pathway, project showcase, basic conversion visibility and future-ready foundation.

## First-stage scope

The first-stage project includes:

- Responsive modern website design and build
- Light digital brand refresh
- Project/gallery structure
- Structured enquiry form
- Light CRM-style enquiry foundation
- Basic enquiry, SEO and conversion visibility
- Future-ready technical foundation
- Launch, testing and handover notes

## Important scope guardrails

Do not expand this project into a full CRM, AI assistant, operational platform or complex marketing system unless Gary explicitly approves a scoped variation.

AI is optional and should be priced separately after the core project is accepted.

Advanced CRM, project tracking, customer portal, payment support, maintenance workflows and business integrations are future opportunities, not MVP commitments.

## MVP complete when

The MVP should be considered complete when:

- Refreshed website is live or ready for launch
- Agreed core pages are built
- Mermaid logo and refreshed digital brand direction are applied
- Project/gallery structure is in place
- Enquiry form is live and tested
- Enquiry data is captured in the agreed format
- Basic enquiry status or review structure is available
- Basic dashboard/reporting view exists
- SEO foundations are applied
- Google Analytics/Search Console setup guidance or connection is completed where access is available
- Client has reviewed and approved final content
- Handover notes are provided
