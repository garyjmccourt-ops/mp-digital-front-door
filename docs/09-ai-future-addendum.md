# 09 — AI Future Addendum

## Position

AI is optional and should be priced separately after the core project is accepted.

Do not include AI build work in the first-stage fixed price unless Gary explicitly approves a scoped variation.

## Possible future AI use cases

- Customer guidance
- Enquiry qualification
- Project readiness prompts
- FAQ support
- Enquiry summaries for internal review
- Suggested next steps based on enquiry type

## Pricing dependencies

AI pricing will depend on:

- Selected AI platform
- API or credit costs
- Usage volume
- Enquiry volume
- Required automation level
- Hosting and integration approach
