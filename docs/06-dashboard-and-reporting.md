# 06 — Dashboard and Reporting

## Purpose

Provide basic visibility of enquiry quality, source, status and conversion indicators.

## First-stage metrics

- Total enquiries received
- New enquiries by period
- Enquiry status
- Enquiry type/project type
- Location or target region where supplied
- Budget readiness where captured
- Timing readiness where captured
- Qualified versus unqualified enquiries
- Won/lost enquiry outcomes where recorded
- Basic website/search visibility indicators where available

## Potential data connections

- Website enquiry records
- Google Analytics
- Google Search Console
- Basic SEO/search performance indicators

## Guardrail

This should be treated as a practical foundation view, not a full business intelligence platform.
