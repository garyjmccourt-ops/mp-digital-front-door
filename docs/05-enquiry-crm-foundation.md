# 05 — Enquiry CRM Foundation

## Purpose

Create a lightweight foundation for structured enquiry capture, review and follow-up.

This is not a full CRM in the MVP.

## Potential enquiry fields

- Name
- Email
- Phone
- Preferred contact method
- Customer type
- Project type
- Location/suburb
- Budget readiness or expected investment range
- Timing
- Property/site status
- Current project stage
- Project notes
- Enquiry source, where available

## Potential statuses

- New enquiry
- Needs review
- Contacted
- Qualified
- Consultation required
- Quoting
- Not suitable
- Won
- Lost

## MVP decision needed

Confirm whether enquiries should write to:

- Email only
- Spreadsheet
- Simple database table
- CRM-style table
- Third-party CRM

## Guardrail

The first stage should not become a full CRM build unless separately approved.
