# 01 — Product Scope

## In scope for first stage

### Website design and build

- Responsive modern website
- Premium visual presentation
- Strong use of project photography
- Clear calls to action
- Improved contact/enquiry pathway
- Basic SEO structure
- Page titles and meta descriptions
- Image naming and alt-text guidance
- Accessible and maintainable page structure

### Light digital brand refresh

- Retain Mermaid Pools’ existing name and brand equity
- Use supplied Mermaid Pools logo and mermaid silhouette assets
- Explore cleaner digital use of the logo
- Consider simplified digital treatment for logo/profile/social use
- Retain water-led blue palette
- Improve typography, spacing, contrast and layout
- Create practical usage rules

### Project gallery

- Modern project/gallery structure
- Project categories
- Use of supplied project photography
- Image selection guidance
- Future-ready structure for case studies

### Enquiry pathway

- Structured enquiry form
- Useful lead qualification fields
- Simple status/review structure
- Practical foundation for follow-up

### Dashboard/reporting foundation

- Total enquiries
- New enquiries by period
- Enquiry status
- Project/enquiry type
- Location where supplied
- Budget/timing readiness where captured
- Qualified vs unqualified enquiries
- Won/lost outcomes where recorded
- Basic website/search visibility indicators where available

## Out of scope unless separately approved

- Full CRM
- AI assistant
- Customer portal
- Project stage tracking
- Quote tracking
- Task management
- Document storage
- Maintenance tracking
- Payment/invoicing workflows
- Business platform integrations
- Professional photography
- Major image editing
